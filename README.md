# UADER_IS2_ALLAUD

## Descripción del Proyecto

Este repositorio corresponde a la materia **Ingeniería de Software II** de la
Universidad Autónoma de Entre Ríos (UADER). Contiene ejercicios prácticos
de gestión de configuración y programación en Python.

---

## Estructura del Repositorio

### Carpetas principales

- **src/** → Código fuente de los programas Python
- **doc/** → Documentación del proyecto
- **bin/** → Archivos ejecutables
- **script/** → Scripts auxiliares

### Herramientas utilizadas

- Git
- Python 3.13
- Pip 26

---

## Programas incluidos

### primos.py

Programa que calcula e imprime los números primos dentro de un rango dado.

#### Instrucciones de uso

1. Clonar el repositorio
2. Navegar a la carpeta `src/`
3. Ejecutar con `python3 primos.py`

#### Requisitos

1. Tener instalado Python 3
2. Tener instalado Git
